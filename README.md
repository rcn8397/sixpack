# sixpack
Six-Pack Cherry MX Macro Pad

# Layout
Six Switch MX Layout

# Plate & Case

 * Switch Type: MX {_t:3}
 * Stabilizer Type: Cherry + Costar {_s:1}
 * Case Type: Sandwich
 * USB Cutout: On
   * Location: 0 (default)
   * Width: 12
 * Mount Holes: 4, 3, 8
 * Edge Padding: On, 10,10,10,10 (W,N,E,S)
 * Plate Corners: On, 4
 * Custom Polygons: Off
 * Kerf: Off
 * Key Unit Size: Off
 * Line Color: Off
 * Line Weight: Off
